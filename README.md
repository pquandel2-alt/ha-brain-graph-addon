# Brain Graph — Home Assistant Add-on Repository

Live 3D-Visualisierung aller Home Assistant Entitäten, Geräte und Automationen als Sidebar-Panel.

## Installation

1. In Home Assistant: **Einstellungen → Add-ons → Add-on Store**
2. Oben rechts ⋮ → **Repositories**
3. URL hinzufügen:
   ```
   https://github.com/pquandel2-alt/ha-brain-graph-addon
   ```
4. Store neu laden, "Brain Graph" installieren und starten
5. Sidebar-Eintrag „Brain Graph" erscheint

Details: [brain_graph/README.md](brain_graph/README.md)

## Voraussetzung

Home Assistant OS oder Supervised (Add-ons werden von HA Core/Container nicht unterstützt).
